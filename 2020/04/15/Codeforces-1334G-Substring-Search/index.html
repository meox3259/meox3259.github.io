Codeforces 1334G - Substring Search

Posted on 2020-04-15 In Competitive Programming

Link:

Codeforces 1334G - Substring Search

Solution:

经典$FFT$问题

搞一个式子
$$(s_i - t_j)^{2}(p_{s_i} - t_j)^{2}$$
那么把$s$反转后和$t$进行上述函数的卷积，如果存在相等则这位上的和等于$0$。
于是拆开后对包含$i$与$j$的项用多项式乘法计算，其他的项可以通过直接计算得出。

时间复杂度$O(nlogn)$
