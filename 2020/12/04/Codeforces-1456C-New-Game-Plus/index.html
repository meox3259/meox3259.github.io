Codeforces 1456C - New Game Plus!

Posted on 2020-12-04 Edited on 2020-12-05 In Competitive Programming

Link:

Codeforces 1456C - New Game Plus!

Solution:

是自己做出来的。

首先正数应该都放在前面，然后后面放负数。
然后在正数后面放负数，直到总和小于$0$，考虑之后和当前这个前缀和怎么放。
维护一个堆，把每次清空的机会看成一个板子，那么每个负数可以挂在一个板子后面，堆中维护的是每个板子后面跟了几个数。
从小到大枚举，每次取出堆中最小次数，然后把当前数挂上去，最后把前缀和挂上去。

时间复杂度$O(nlogn)$。
