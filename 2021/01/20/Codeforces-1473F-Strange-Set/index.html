Codeforces 1473F - Strange Set

Solution:

套路题。

是一个比较经典的最大权闭合子图，先把正的贡献加上，然后建图，源连向正的贡献，负的贡献连向汇，然后有限制关系的连上容量为$inf$的边。
但是这样边数太多，会爆空间，考虑如果$i$与$j$联通，$j$与$k$联通，那么$i$和$k$没必要连边，这里连边的时候用$bitset$判一下即可。

时间复杂度$O(\frac{n^3}{64}+dinic)$。
