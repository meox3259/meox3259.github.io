GYM102483A - Access Points

Posted on 2021-08-25 In Competitive Programming

GYM102483A - Access Points

Solution:

首先发现$x$和$y$两维是可以分开计算的。

于是问题变成了给定序列$a$，求一个不降的序列$b$使得$\sum_{i=1}^{n}{(a_i-b_i)^2}$最小。

观察一下可以发现，最后的答案肯定是阶梯状的。将$a$划分成几段，并且每段中的数都是一样的，为这段的平均数(对于一段下降区间，取平均数最优)。这样可以$O(n^2)$解决。

考虑加速这个过程。利用单调栈去维护，如果新加入的元素的平均值比栈的末尾小，那么合并，直到不能合并为止。为什么是对的，感性理解一下如果当前尾部的元素较小，那么只能合并到前一段或后一段。那么由于题目要求前一段的平均值比后一段小，所以放入前一段更优。
