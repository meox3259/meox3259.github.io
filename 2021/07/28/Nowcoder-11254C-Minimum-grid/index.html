Nowcoder 11254C - Minimum grid

Posted on 2021-07-28 In Competitive Programming

Nowcoder 11254C - Minimum grid

Solution:

首先考虑如何填使得方案合法，一个简单的想法是对于每个可以填的位置$(X,Y)$，填上$min(A[X],B[Y])$。这样虽然合法，但是可能在一些不影响的位置填了多余的数。

仔细思考一下，从小到大考虑删去一些数，删去一种数不会对其他种类的数造成影响，所以每种数是独立的问题。那么对于一种数如果出现了$X$次，那么对于上面的构造方案至少会在$X$个位置填上这个数。减少的情况是在行与列交叉的位置共用一个数，这里用网络流解决即可。

时间复杂度$O(N^2M)$。由于网络流所以具体复杂度未知。
