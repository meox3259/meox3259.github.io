KEYENCE Programming Contest 2021 D - Choosing Up Sides

Posted on 2021-03-03 Edited on 2021-03-05 In Competitive Programming

Link:

KEYENCE Programming Contest 2021 D - Choosing Up Sides

Solution:

有迹可循的构造题。首先要确定最少的次数，考虑列等式算一算，可以得出这样的式子:
$$(n+m)M^2=m\tbinom{2M}{2}$$
可以得出$n:m=M-1:M$，其中$M=2^{N-1}$。
那么$n+m$至少为$2^N-1$，当然其实$2^N-1$就行。
然后考虑构造，比赛的时候也卡在了这里，首先构造需要满足两个要求，一个是$n:m=M-1:M$，另一个是每个人都相同。
由于二进制空间下$0$和$1$是等价的，因为把$0$和$1$反转没有区别，所以考虑利用二进制表示每个人。
由于轮数有$2^N-1$轮，缺少了$0$，所以可以按$popcount(S)$的奇偶进行分类。
对于第$i$轮，第$j$个人被分配在$popcount(iandj)and1$组，可以发现上面进行奇偶分类后每类的人都是等价的，所以满足条件。

时间复杂度$O(4^n)$。
