Codeforces 1493F - Enchanted Matrix

Posted on 2021-09-26 Edited on 2021-09-27 In Competitive Programming

Codeforces 1493F - Enchanted Matrix

Solution:

首先可以发现行和列是独立的，于是可以把行和列分开算出方案然后乘起来得出答案。

考虑计算行的答案，列也用相同的方法。对于每个质因子独立地计算出能够最多乘几个仍然是行的一个周期。

由于已知$n=\prod{p_{i}^{a_i}}$是一个循环节，可知对于任意长度$\leq n$的循环节$r$都有$r \mid n$。

于是考虑试除每个$p_i$，对于当前的循环节$n$如果$\frac{n}{p_i}$是循环节的话说明$p_i$也是循环节。于是可以有$n=\frac{n}{p_i}$。

现在考虑如何通过操作判断一个长度$r$是否为循环节，那么等价于问$\frac{n}{r}$是否为循环节。设$l=\frac{n}{r}$，也就是可以把字符串均分为$r$段相等的长为$r$的子串。如果不考虑询问要求不相交，那么直接询问$[1,r-1]$和$[2,r]$是否相等即可，如果相等说明$\frac{n}{r}$以及$r$是一个循环节。

但是现在不允许相交，考虑一个替代的方式。设$k=\lceil \frac{r}{2} \rceil$，这里只需要询问$[1,k-1]$和$[k,r-1]$，$[1,k-1]$和$[k + 1, r]$是否相等即可。这里的意义在于取一个中介来判断两个相交的段是否相等，于是可以通过两次询问得出 一个长度是否为循环节。

于是通过这样的操作，即可以算出一行或一列的方案，最后乘起来即可。

时间复杂度$O(nlogn+mlogm)$。
