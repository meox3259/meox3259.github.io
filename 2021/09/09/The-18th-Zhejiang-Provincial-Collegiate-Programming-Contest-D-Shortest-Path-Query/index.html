The 18th Zhejiang Provincial Collegiate Programming Contest D - Shortest Path Query

Posted on 2021-09-09 In Competitive Programming

Link:

The 18th Zhejiang Provincial Collegiate Programming Contest D - Shortest Path Query

Solution:

题目的特性在于每条边连接二进制串以及二进制串的前缀，考虑如何利用这个性质解决问题。

可以发现，对于一个二进制串，它的「祖先」最多有$logn$个。并且对于每个点，不存在一条边直接从这个点的左子树连向右子树，这也说明如果想从左子树走到右子树需要走到这个点或这个点的祖先。

那么由于上面的性质，对于$s$和$t$两个点，肯定会经过$s$和$t$的公共祖先。由于公共祖先不超过$logn$个，所以枚举公共祖先并计算公共祖先到$s$和$t$的最短距离即可算出最短路径。

现在考虑如何计算出每个点到所有祖先的最短路径。考虑对于求出每个点到子树中的所有点的最短路。由于一个点的祖先不超过$logn$个，那么说明每个点最多在$logn$个点的子树中。于是对每个点暴力跑最短路即可。

时间复杂度$O(nlog^2n)$。
